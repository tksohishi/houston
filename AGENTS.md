# Houston

Discord-based command center for AI coding agents. Supports Claude Code, Codex CLI, and Gemini CLI.

## Commands

- `bun dev` — run with `--watch` and verbose logging (logs written to `tmp/dev.log`)
- `bun test` — run tests
- `bun run typecheck` — typecheck

Always run both `bun test` and `bun run typecheck` to verify changes.
